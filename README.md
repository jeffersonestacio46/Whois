# Pesquisa de Whois

**Esse script foi escrito interamente em Python**

Esse script realiza pesquisas de informações Whois publicas, colocando apenas o link do site para realização de pesquisas.

Para utilização no seu Kali linux, segue os comandos a baixo:

> `Python3 Whois.py3` Lembrando em usar esse comando no terminal dentro da pasta onde foi salvo o arquivo.

Use com responsabilidade
